== Description ==

Das „Mein Seh-Check“-Plugin ermöglicht es Abonnenten von Mein Seh-Check, Online-Sehtests als Block in den Gutenberg Editor einzufügen.

Alternativ kann bei Page Buildern wie Avada, Elementor, Divi, Oxygen etc. folgender Shortcode verwendet werden:

`[mein-seh-check kd-nr="XXX-XXX-XXXX"]`

Für die Verwendung des Plugins ist ein Abonnement auf [Mein-Seh-Check.de](https://www.mein-seh-check.de) zwingend notwendig.

== Frequently Asked Questions ==

= Warum wird kein Seh-Check angezeigt (Fehler 403)? =

Damit der Seh-Check auf Ihrer Website eingeblendet werden kann, müssen Sie im Login Bereich von Mein-Seh-Check.de unter dem Menüpunkt „Profil > Persönliche Daten“ das Feld „Website“ mit der Domain der Website versehen, auf der die Seh-Checks integriert werden sollen.

= Wo finde ich meine Kundennummer? =

Die Kundennummer findet man im Login Bereich von Mein-Seh-Check.de unten Links in der Seitenleiste unter dem Account-Namen und im Menüpunkt „Einbettung“, nachdem man bei „Art der Website“ die Option „WordPress“ ausgewählt hat und dann auf den Button mit der Beschriftung „Weiter“ klickt.
